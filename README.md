# Heltec-LoRa-case-with-battery
## This repo contains the design for a printable case for the Heltec LoRa


### Source folder contents

- Solidworks parts
- Solidworks entire
- STL files
 
### Print parameters

- Preferable to use PETG
- Due to the reduced thickness, 100% filling is recommended.
